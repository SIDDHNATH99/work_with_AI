# Python Backend Executable Builder (Claude Skill)

## Overview
This Claude skill helps automate building a **Windows executable (.exe)** from a Python backend application using **PyInstaller**.

## What It Does
The skill assists with:

- Building a Python application into a Windows executable
- Packaging the application using PyInstaller
- Creating versioned releases
- Preparing the application for deployment

## When This Skill Is Used
This skill is triggered when users want to:

- build a release
- create an executable
- package a Python application
- deploy a new version of a backend service

## Purpose
The goal of this skill is to simplify the process of converting Python backend applications into **deployable Windows executables**.

Part of my **Claude AI learning experiments** exploring how Claude Skills can automate common developer workflows.
