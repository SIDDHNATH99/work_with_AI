# Budgeting Calculator (Claude Artifact)

This project demonstrates a simple budgeting tool created using Claude Artifacts.

## Features
- Enter income
- Add expenses
- Calculate remaining balance
- View Charts and Visualizations of expenses and incomes

## Purpose
The goal of this experiment was to understand how Claude Artifacts can generate simple interactive tools.

## Claude Feature Used
Artifacts

## Learning Outcome
Claude can generate functional UI tools from structured prompts.

## Prompt used to generate the tool
Create One budgeting tool, Add income & expenses, Budget categories, Charts & visualizations, Monthly summary in Professional and finance-like style
